# Lpo Ftl (Лэпо Фэтэл)

Lpo Ftl: инженерный язык Лепориш с однозначными сокращениями (примерно в 2 раза короче английского с тем же алфавитом) + игра-симулятор фэнтези мира «Толстая Ящерица» с синтезом речи на основе языка Лепориш.

![флаг.png](Lpo_flg.png)

## Как использовать:
Основные файлы для использования языка:
- **«0 Lpo.txt»**: словарь с описанием правил, с примерами, леммы отсортированы по семантическим категориям. Язык намеренно ограничен однобайтовой кодировкой Win1251. Если не можете открыть в этой кодировке, используйте файл **«0 Lpo UTF8.txt»**.
- **«0 Lpo.png»**: геометрия алфавита и служебных символов.
- **«An.ttf»**: шрифт с нужной геометрией символов. Выровнен по пиксельной сетке для 9pt в 96DPI. Также есть моноширинный **«An1.ttf»** и более консервативный **«Run.ttf»**.

## Программа-ридер для Windows:
**«Lpo Vok.exe»**: работает с файлом **«0 Lpo.txt»** в локальной папке и использует установленный шрифт **«An»**.
Исходный код представлен единым файлом **«Lpo Vok.cpp»** в папке **«tuli»**, WinAPI C++.

### перевод меню настроек (**«st»**):
- **«uv ol okn»** = поверх всех окон
- **«oi fh vd»** = только целое слово
- **«xrb sns»** = чувствительность к регистру
- **«rld vok»** = перезагрузить словарь

## [Программа-ридер для Android](/Lpo_Android):
APK файл — сама программа (цифровой подписи нет, извините). В архиве основные части исходного кода (Kotlin, Android Studio).
В эту программу нужно загружать UTF8 версию словаря.

### перевод меню настроек:
- **«lod vok»** = загрузить словарь
- **«rld vok»** = перезагрузить словарь
- **«oi fh vd»** = только целое слово
- **«xrb sns»** = чувствительность к регистру
- **«iho»** = выход

## [Инструменты в папке «tuli»](/tuli):
Здесь инструменты разработчика и коды программ анализа и редактирования словаря.
Программы читают словарь из **«1.txt»** в локальной папке.
Основные файлы:
- **«1 qk vok fom.cpp»** — проверяет, есть ли в строках лемм разделитель **«==»**, и дополняет последнее **«=»**.
- **«2 omon & vd em.cpp»** — проверяет нарицательные слова и их алломорфы на наличие омонимов, подсчитывает общее число лемм и алломорфов.
- **«3 nim em.cpp»** — подсчитывает общее количество лемм.
- **«4 lbl bzd kmbi do 3bkv.cpp»** — отмечает занятые комбинации слов из 1..3 букв в файле **«2.txt»**.
- **«5 bv stt.cpp»** — подсчитывает статистику использования букв алфавита во всех леммах и алломорфах.
- **«Lpo.h»** — содержит набор макросов ключевых выражений C++, заменяя их на слова Лепориша.

## [Фэтэл (Ftl, Толстая ящерица)](/Ftl):
Дизайн-документы игры в жанре фэнтези-симулятор жизни. Планируемые особенности:
- Открытый мир-песочница с ивентами, небольшими поселениями, строительством и физической экономикой.
- Управление в основном режиме за одного персонажа от первого/третьего лица, управление отрядом.
- Редактор флоры и фауны.
- Охота на монстров, защита поселения.
- Синтез голоса на языке Лепориш, возможно, примитивный ИИ, понимающий текст на ограниченном Лепорише в свободной форме.
- Возможность полностью переключить интерфейс на Лепориш.
